Assessing Vegetation Restructuring and Wildfire Resilience in Freetown-Fall River State Forest

Using LiDAR technologies and LANDSAT imagery

Team Members:

Aaron Richmond-Crosset, Angela Ruan, Caleb Kluchman

Affiliation:

GEOG 382: Advanced Remote Sensing

Project Description:

This project assessed the effectiveness of prescribed burning and timber harvesting on vegetation structure and wildfire resilience in Freetown-Fall River State Forest using multi-temporal LiDAR (2011, 2021) and NDVI analysis (2003–2024). Results showed that LiDAR effectively detected fine-scale canopy changes, such as narrow burn corridors, while NDVI captured broader, more recent vegetation dynamics. The analysis revealed post-burn regrowth and localized canopy loss, demonstrating the value of integrating both remote sensing methods to monitor forest management outcomes and improve wildfire risk reduction strategies.
